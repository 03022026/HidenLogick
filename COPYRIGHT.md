## Anti-Plagiarism & Anti-Theft Protection

### Prohibited Actions
The following actions are **strictly forbidden**:

1. **Copyright Removal**
   - Deleting, modifying, or obscuring copyright notices
   - Removing author attribution from files
   - Stripping license headers from code

2. **False Attribution**
   - Claiming to be the original author
   - Presenting code as entirely original when it uses HidenLogick components
   - Removing references to the original project

3. **Misrepresentation**
   - Claiming HidenLogick is your proprietary software
   - Selling HidenLogick code without proper attribution
   - Implying official affiliation with HidenLogick when none exists

4. **Unauthorized Commercialization**
   - Selling unmodified HidenLogick without attribution
   - Using HidenLogick as a "white-label" solution without disclosure
   - Hiding the open-source nature of the software

5. **Plagiarism**
   - Copying entire modules without attribution
   - Renaming files/functions to hide origin
   - Publishing as part of a different project without credit

### Community Harm
Violations protect against:
- Developers claiming others' work
- Commercial exploitation of open-source code
- Obfuscation of software origins
- Loss of community recognition for contributors

## Enforcement

### For Repository Hosts (GitHub, PyPI, etc.)
We reserve the right to file DMCA takedown notices against:
- Repositories that remove copyright notices
- Packages claiming false authorship
- Code that misrepresents its origin

### For Derivative Works
Creators of forks, mods, or extensions must:
1. Retain ALL original copyright notices
2. Clearly mark modifications as such
3. Provide attribution: "Based on HidenLogick by [original author]"
4. Link to the original project
5. Use a distinctly different name

**Example Compliance:**
```
MyMC Launcher - An enhanced version of HidenLogick
Original by: [HidenLogick creators]
Repository: [link to HidenLogick]
Modifications: [describe your changes]
```

### For Commercial Use
Commercial use is permitted, but requires:
- Prominent attribution to HidenLogick
- Clear disclosure that it's based on open-source code
- Link to original repository
- Compliance with MIT License terms

## Attribution Examples

### ✅ CORRECT
- "HidenLogick Launcher - An enhanced Minecraft launcher based on the open-source HidenLogick project"
- "MyLauncher, powered by HidenLogick"
- "This software uses components from HidenLogick (MIT License)"

### ❌ INCORRECT
- "HidenLogick" (without attribution)
- "Original Minecraft launcher (all work by me)"
- "TLauncher Alternative" (hiding HidenLogick origin)
- Using the exact name "HidenLogick" for non-derived work
